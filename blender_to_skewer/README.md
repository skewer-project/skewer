# Blender to Skewer

This is a script that is run *inside of blender* to export a blender scene into a scene json file consumed by Skewer.

To use this, open blender with a scene loaded in, navigate to the "scripts" tab, and open the `blender_export.py` python script.

You should then be able to execute the script inside of blender, and have a full scene.json file along with assets saved to your downloads folder
